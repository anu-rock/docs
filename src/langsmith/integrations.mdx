LangSmith provides support for [LangChain](/oss/python/langchain/overview) and [LangGraph](/oss/python/langgraph/overview) as well as integrations with a growing set of popular [LLM providers](#llm-providers) and [agent frameworks](#agent-frameworks). This overview highlights some of the integrations in each category, while the complete list of supported integrations have dedicated guide pages in the navigation bar for setup and usage.
