---
title: Chat models
---

[Chat models](/oss/langchain/models) are language models that use a sequence of [messages](/oss/langchain/messages) as inputs and return messages as outputs (as opposed to using plain text). These are generally newer models.

<Info>
If you'd like to write your own chat model, see [this how-to](/oss/how-to/custom_chat). If you'd like to contribute an integration, see [Contributing integrations](/oss/contributing#add-a-new-integration).
</Info>

<Tabs>
<Tab title="Groq">

#### Install dependencies

<Tip>
See [this section for general instructions on installing LangChain packages](/oss/langchain/install).
</Tip>

<Tabs>
<Tab title="npm">

```bash
npm i @langchain/groq
```

</Tab>
<Tab title="yarn">

```bash
yarn add @langchain/groq
```

</Tab>
<Tab title="pnpm">

```bash
pnpm add @langchain/groq
```

</Tab>
</Tabs>

#### Add environment variables

```bash
GROQ_API_KEY=your-api-key
```

#### Instantiate the model

```typescript
import { ChatGroq } from "@langchain/groq";

const model = new ChatGroq({
  model: "llama-3.3-70b-versatile",
  temperature: 0
});
```
