To install the core LangChain package

:::python
```bash
pip install -U langchain
```
:::

:::js
```bash
npm install langchain
```
:::

LangChain also provides integrations to 100s of LLMs and thousands of other integrations.
To avoid package bloat, these live in independent provider packages.
In order to install the package for the provider you would like to work with,
see the integrations page.
